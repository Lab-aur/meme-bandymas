# MemeGeneratorius

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.4. It can be installed by running command in your terminal: npm install -g @angular/cli


#Installing

To install all dependencies run 'npm install' command.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
